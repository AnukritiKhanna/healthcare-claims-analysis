# Healthcare Claims Data Analysis

## Objective
Analyze healthcare claims data to identify cost drivers, utilization patterns, and trends.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Excel

## Key Goals
- Clean and preprocess healthcare claims data
- Perform exploratory data analysis
- Identify high-cost procedures and utilization trends
- Generate actionable insights
